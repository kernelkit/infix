==== Test sequence
. Initialize
. Configure VLAN 10 on target:data with IP 10.0.0.2
. Waiting for links to come up
. Ping 10.0.0.2 from VLAN 10 on host:data with IP 10.0.0.1
. Remove VLAN interface, and test again (should not be able to ping)
